# Skill: Style Research & Exploration

This repository contains the evolutionary history of the BG3 Splendor visual style.

## Purpose
Use these scripts if you want to experiment with new visual themes or perform quality audits on the existing aesthetic.

## Contents
- **Style Exploration**: `test_modular_prompts.py` and `test_style_exploration.py` contain the prompt engineering history.
- **Samples**: `generate_tier_samples.py` was used to establish the visual difference between Tier 1, 2, and 3 cards.
- **Attributes**: `generate_attribute_samples.py` explores how different resource types (Radiant, Arcane, etc.) should feel visually.

## Reference
These scripts are preserved to ensure that if the project aesthetic ever drifts, we can return to the original "Source of Truth" styles.

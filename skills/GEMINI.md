# Skills Directory Guidelines

This directory contains executable, reusable agentic workflows (Skills).

- **Structure**: Every skill must have its own subdirectory.
- **Documentation**: Each skill must have a `SKILL.md` explaining its purpose and a `STANDARDS.md` defining its coding rules.
- **Execution**: Scripts meant to be executed by agents must reside within `scripts/` inside their respective skill folder.
- **Generalization**: No hardcoded one-off scripts. Scripts must be configuration-driven and reusable.
